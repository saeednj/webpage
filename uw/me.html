My Name

My name is Saeed. It is an Arabic word, but commonly used in Iran as a first name. In Arabic it means happy/merry, but in Iran we use it with the meaning of blessed. In native script it is written like: سعید
and it is pronounced like [s@ i:d] in IPA ASCII phonetics (my last name is also read as [ne dZa:ti]).
